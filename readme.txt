This plugin allows you to insert/append custom messages into your RSS feed.

== Description ==

This Plugin allows you to insert custom messages at the start or the end of your blog entry in your RSS feed. Can be useful to put in source links if proper attribution is not being given to your content which is syndicated. You can also put in a small logo or icon of your blog which could link back to your site.

You can specify the custom text or HTML that you want to add by going to the settings page at : Settings > Message In RSS Feed.

== Installation ==

Installation steps : 

- Unzip the contents of the Zip file into your wp-content/plugins/ directory

- Login to your admin panel and then activate the plugin.

== Usage ==

Just go to your Settings > Message In RSS Feed and configure what you would want to insert and where it should be inserted.

== Frequently Asked Questions ==

= How is this useful =

You can insert your copyrights and attribution links if proper attribution is not being given by people who are syndicating your RSS feed.

= Can I use HTML =

Yes you can use HTML too. You can beautify your RSS feed by adding your blog logo icon in the RSS feed. Moreover you can also make it link back to your main page.
